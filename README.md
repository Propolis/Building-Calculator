# Building Calculator — QGIS Plugin

Плагин для расчёта количества жителей и парковочных мест на основе выбранного полигона здания.

## Установка

### macOS

1. Скопируйте папку `building_calculator` в директорию плагинов QGIS:
   ```bash
   cp -r building_calculator ~/Library/Application\ Support/QGIS/QGIS3/profiles/default/python/plugins/
   ```

2. Откройте QGIS

3. Включите плагин:
   - **Plugins → Manage and Install Plugins**
   - Найдите **Building Calculator**
   - Поставьте галочку для активации

4. Иконка плагина появится на панели инструментов

### Windows

1. Скопируйте папку `building_calculator` в:
   ```
   C:\Users\<ИМЯ_ПОЛЬЗОВАТЕЛЯ>\AppData\Roaming\QGIS\QGIS3\profiles\default\python\plugins\
   ```

2. Откройте QGIS и включите плагин (см. шаг 3 выше)

## Использование

1. Создайте или откройте полигональный слой
2. Нарисуйте полигон здания
3. Выделите полигон (один)
4. Нажмите кнопку **Building Calculator** на панели инструментов
5. Укажите количество этажей и параметры расчёта
6. Получите результат: квартиры, жители, парковочные места

## Параметры расчёта

- **Этажи** — количество этажей в здании
- **Площадь квартиры** — средняя площадь одной квартиры
- **Жители** — расчёт по количеству на квартиру или по м² на жителя
- **Парковка** — расчёт на квартиру, на жителей или на м²
